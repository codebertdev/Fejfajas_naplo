Fejfájás_naplo – Python GUI alkalmazás

Ez a projekt egy egyszerű fejfájásnapló alkalmazás Pythonban, amelyben a felhasználó rögzítheti a napi fejfájás erősségét, megjegyzéseit, visszanézheti a korábbi bejegyzéseket, valamint átlagot számolhat.
A felület tkinter segítségével készült.

Modulok

Tanult modulok:

math

random

Bemutatandó modul:

tkinter + ttk
(GUI elemek, gombok, szövegbeviteli mezők, események)

Saját modul:

sri_tracker_utils.py

SriHeadacheEntry osztály

sri_calculate_average() függvény

Fájlok

main.py – a program belépési pontja

app.py – GUI logika

sri_tracker_utils.py – saját modul (osztály + függvény)

Futtatás
python main.py
